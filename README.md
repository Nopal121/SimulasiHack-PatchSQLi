# 🔐 SQL Injection Lab - Hack & Patch

Sebuah laboratorium pembelajaran untuk memahami **SQL Injection** dan cara memperbaikinya menggunakan **Prepared Statements** (Parameterized Queries). Proyek ini ditujukan untuk keperluan pendidikan dan praktikum keamanan siber.

## 📋 Deskripsi

Proyek ini mendemonstrasikan:
1. **Kerentanan SQL Injection** - Login bisa ditembus menggunakan payload `' OR '1'='1`
2. **Teknik Patch** - Memperbaiki kode menggunakan Prepared Statements
3. **Pengujian Ulang** - Membuktikan bahwa patch berhasil memblokir serangan

## 🛠️ Prasyarat

- **XAMPP** (Apache + MySQL) atau web server dengan PHP & MySQL
- PHP versi 7.0 atau lebih baru
- MySQL versi 5.6 atau lebih baru
- Browser modern
